replaceText
filter in module cat.filters

Description

Replaces text passages with other text, based on regular expressions

Usage

In HTML Template Binding

{{ replaceText_expression | replaceText:pattern:options:replacement }}

In JavaScript

$filter('replaceText')(text, pattern, options, replacement)

Parameters

ParamTypeDetails
textstring

original text

patternstring

regular expression

optionsobject

regular expression options

replacementstring

replacement text
