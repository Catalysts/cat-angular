CatBaseTabsController
controller in module cat.controller.base

Description

The base code for handling sub entites (as tabs). Includes the instantiation of one controller per tab/list and lazy loading of the approrpiate data

Usage

CatBaseTabsController($scope, $controller, $stateParams, $location, catElementVisibilityService, config);

Parameters

ParamTypeDetails
$scopeObject

The angular scope

$controllerObject

The angular $controller service

$stateParamsObject

The ui-router $stateParams service

$locationObject

The angular $location service

catElementVisibilityServiceObject

The visibility service to check wheter or not a tab should be rendered

configObject

The config as handled by state resolve
